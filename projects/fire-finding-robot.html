Fire-Finding Rescue Robot

Autonomous Navigation & Path Planning | March 2025

Challenge

[cite_start]

Traditional path-planning algorithms often fail in dynamic environments[cite: 20]. [cite_start]This project required the implementation of an adaptive algorithm capable of real-time replanning while optimizing for dual objectives: rescuing a survivor and navigating to a fire location[cite: 20, 29].

Technical Implementation

[cite_start]

At the core of the system is the D* Lite algorithm, which allows for efficient dynamic path planning via backward tree construction[cite: 23].

    [cite_start]
  • Algorithm Design: Utilized a Chebyshev distance heuristic optimized for 8-directional movement, allowing for realistic diagonal travel costs of $\sqrt{2}$[cite: 24, 30].
    • [cite_start]
  • Environment Dynamics: The robot operated in a grid-based environment with randomly shuffled obstacles to simulate unpredictable disaster conditions[cite: 28].
    • [cite_start]
  • Optimization: Managed complexity through priority queue sorting based on total cost ($cost2Reach + cost2Go$) and node status tracking to prevent redundant processing[cite: 32, 33].

Key Lessons & Results

[cite_start]

The project successfully demonstrated that D* Lite's recomputation efficiency is vastly superior to A* for dynamic environments[cite: 35, 37]. [cite_start]We found that algorithm sensitivity to heuristic choice and move costs significantly impacts path optimality[cite: 36].
